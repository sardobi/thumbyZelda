# Zelda clone for Thumby

## Dev env setup

I use the [Thumby code editor](https://code.thumby.us/).

To get set up:
- Connect your Thumby to the editor
- Choose `UTILITIES` > `MAKE NEW GAME`
- Enter `Zelda.py` as the name of the main project file
- Paste or import `Zelda.py` from this repo into your `Zelda.py`
- For each other Python `.py` file in this repo:
  - Create a new editor tab (`FILE` > `NEW TAB`)
  - Set the path: `FILE` > `SET PATH`, then enter `/Games/Zelda/<name of the file>.py`

To get the editor's emulator working:
- Tick the white and red checkboxes for the `Zelda.py` editor
- Tick the white checkboxes for the rest
